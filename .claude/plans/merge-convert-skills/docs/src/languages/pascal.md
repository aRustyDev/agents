# Pascal

> Structured programming language designed for teaching and system development.

## Overview

Pascal is a procedural programming language designed by Niklaus Wirth in 1970 for teaching structured programming and data structuring. It influenced many subsequent languages and evolved into Object Pascal (Delphi), which remains used for Windows application development.

## Family Classification

| Dimension | Classification | Notes |
|-----------|----------------|-------|
| Primary Family | Procedural | Structured, strongly typed |
| Subtype | educational | Teaching-oriented design |

## Key Characteristics

- **Strong static typing** with explicit declarations
- **Structured programming** constructs (no goto needed)
- **Nested procedures** with lexical scoping
- **Record types** for structured data
- **Pointer types** with explicit allocation

## Dialects

| Dialect | Notes |
|---------|-------|
| Standard Pascal | ISO 7185, minimal |
| Turbo Pascal | Borland extensions, units |
| Object Pascal | OOP extensions, Delphi |
| Free Pascal | Open source, cross-platform |

## Conversion Notes

- begin/end blocks → braces or indentation
- Procedure/function distinction → unified functions
- Variant records → tagged unions
- Units → modules
- String handling differs from modern languages
