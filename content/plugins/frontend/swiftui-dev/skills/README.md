# Skills

Place skill directories here. Each skill should have a `SKILL.md` file.

## Structure

```text
skills/
└── my-skill/
    ├── SKILL.md      # Main skill definition
    └── assets/       # Optional supporting files
```

## Example

See `content/plugins/homebrew-dev/skills/pkgmgr-homebrew-formula-dev/` for reference.
