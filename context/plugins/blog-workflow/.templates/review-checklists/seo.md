# SEO Review Checklist

Use this checklist to evaluate SEO optimization of a blog draft.

## Title

- [ ] Length under 60 characters
- [ ] Contains primary keyword
- [ ] Compelling/clickable (creates curiosity or promises value)
- [ ] Front-loads important words

## Description

- [ ] Length 150-160 characters (optimal for SERP display)
- [ ] Contains primary keyword naturally
- [ ] Summarizes value proposition clearly
- [ ] Includes call-to-action or benefit

## Headings

- [ ] H1 matches or closely approximates title
- [ ] H2s contain secondary keywords
- [ ] Logical hierarchy (no skipped levels: H1 -> H2 -> H3)
- [ ] Headings are descriptive, not generic ("Introduction", "Conclusion")

## Content

- [ ] Primary keyword appears in first 100 words
- [ ] Primary keyword density 1-2% (not over-optimized)
- [ ] Related terms and synonyms used naturally
- [ ] Internal links to relevant posts present
- [ ] External links to authoritative sources (where appropriate)
- [ ] Content length appropriate for topic (typically 1000+ words for tutorials)

## Images

- [ ] All images have descriptive alt text
- [ ] Alt text includes keywords where natural
- [ ] Filenames are descriptive (not IMG_1234.png)
- [ ] Images are optimized for web (reasonable file sizes)
- [ ] Featured image (ogImage) is set

## URL/Slug

- [ ] Slug is concise and descriptive
- [ ] Contains primary keyword
- [ ] Uses hyphens, not underscores
- [ ] No special characters or encoded spaces

## Technical

- [ ] `canonicalUrl` set if content appears elsewhere
- [ ] `tags` include relevant keywords (3-7 tags recommended)
- [ ] `pubDatetime` is valid ISO 8601
- [ ] No duplicate content concerns

## Scoring Guide

| Score | Criteria |
|-------|----------|
| Excellent | All items pass, well-optimized |
| Good | 1-2 minor issues (warnings) |
| Needs Work | 3+ issues or any critical fails |
| Poor | Multiple critical issues |

## Common Issues

| Issue | Impact | Fix |
|-------|--------|-----|
| Title too long | Truncated in SERP | Shorten to <60 chars |
| Description too short | Missed opportunity | Expand to 150-160 chars |
| Missing alt text | Accessibility + SEO | Add descriptive alt |
| No internal links | Poor site structure | Link to related posts |
| Keyword stuffing | Penalty risk | Use naturally, vary terms |
| Generic headings | Missed keyword opportunity | Make descriptive |
