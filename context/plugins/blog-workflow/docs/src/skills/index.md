# Skills

Skills are reusable knowledge modules that provide guidance on specific topics.

## Available Skills

The blog-workflow plugin includes these skills:

| Skill | Purpose |
|-------|---------|
| [Code Example Best Practices](./code-examples.md) | Writing effective code examples |
| [Content Structure Patterns](./content-structure.md) | Organizing content effectively |
| [SEO for Developers](./seo.md) | Technical SEO fundamentals |
| [Technical Writing Style](./technical-writing.md) | Clear technical communication |

## What is a Skill?

A skill is a focused knowledge module that:

- Addresses a specific domain
- Provides actionable guidance
- Can be referenced during writing
- Helps maintain consistency

## Using Skills

Skills are referenced during workflow phases:

### During Research

Skills inform what to look for and how to evaluate sources.

### During Writing

Skills guide structure, style, and technical accuracy.

### During Review

Skills provide criteria for quality assessment.

## Skill Structure

Each skill includes:

1. **Purpose** - What the skill addresses
2. **Key Concepts** - Core ideas to understand
3. **Best Practices** - Actionable guidance
4. **Examples** - Good and bad patterns
5. **Checklist** - Quick reference during review

## Applying Skills

Reference skills in post frontmatter:

```yaml
skills-applied:
  - code-example-best-practices
  - seo-for-developers
```

This helps reviewers know which standards apply.
